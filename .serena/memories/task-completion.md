# Task Completion Checklist

When a coding task is done: (1) Run `npm run format`. (2) Run `npm run lint` and fix issues. (3) Run `npm run typecheck`. (4) Run `npm run test`. Do not add biome-ignore or ts-ignore without justification; document in code. Prefer E2E tests that verify spec scenarios (e.g. load MP4, play, see video/audio) over low-value unit tests.
